# House Price Prediction Project

## Objective

The goal of this project is to predict house prices using machine learning.

## Dataset

Housing dataset loaded from CSV file.

## Model Used

Linear Regression

## Features Used

* Area
* Bedrooms
* Bathrooms
* Floors
* Year built

## Evaluation Metrics

* MAE
* RMSE

## Results

The predicted house prices were close to the actual prices.
